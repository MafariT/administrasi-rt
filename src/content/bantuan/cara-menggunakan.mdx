## Selamat Datang di Sistem Administrasi RT Digital!

Layanan ini dirancang untuk mempermudah Anda dalam mendapatkan surat pengantar dari Ketua RT. Prosesnya sepenuhnya digital dan dapat diakses kapan saja. Berikut adalah panduan langkah demi langkah untuk menggunakan sistem ini.

### Pendaftaran Data Diri

Sebelum dapat mengajukan surat, Anda harus mendaftarkan data diri Anda terlebih dahulu ke dalam sistem. Ini adalah proses satu kali.

1.  **Buka Halaman Pendaftaran:** Kunjungi halaman **[Daftar Sebagai Warga](/daftar)**.
2.  **Isi Formulir:** Lengkapi semua kolom dengan data yang valid sesuai dengan KTP (Kartu Tanda Penduduk) dan KK (Kartu Keluarga) Anda.
3.  **Unggah Berkas:** Unggah foto atau hasil scan KTP dan KK Anda. Berkas ini diperlukan oleh Ketua RT untuk proses verifikasi.
4.  **Kirim Data:** Setelah semua data terisi, klik tombol "Kirim Pendaftaran".

Status pendaftaran Anda sekarang adalah **"Menunggu Verifikasi"**.

### Proses Verifikasi oleh Ketua RT

Setelah Anda mengirimkan data, Ketua RT akan menerima notifikasi. Ketua RT akan melakukan verifikasi data Anda secara _offline_ dengan mencocokkannya dengan catatan kependudukan resmi yang dimiliki.

Proses ini mungkin memerlukan waktu. Anda dapat membaca lebih lanjut tentang [Proses Verifikasi](/bantuan/proses-verifikasi).

### Mengajukan Surat Pengantar

Setelah data Anda berhasil diverifikasi oleh Ketua RT (status Anda menjadi **"Terdaftar"**), Anda dapat mulai mengajukan surat.

1.  **Buka Halaman Ajukan Surat:** Kunjungi halaman **[Ajukan Surat](/surat)**.
2.  **Verifikasi NIK:** Masukkan 16 digit NIK Anda untuk mengonfirmasi identitas Anda.
3.  **Isi Formulir Surat:** Pilih jenis surat yang Anda butuhkan dan jelaskan keperluannya.
4.  **Kirim Pengajuan:** Klik tombol "Ajukan Surat".

### Cek Status dan Unduh Surat

Anda dapat memantau status pengajuan surat Anda kapan saja.

1.  **Buka Halaman Cek Status:** Kunjungi halaman **[Cek Status](/surat/cek-status)**.
2.  **Masukkan NIK Anda:** Sistem akan menampilkan riwayat semua pengajuan surat Anda.
3.  **Lihat Status:** Anda akan melihat status setiap surat (`Pending`, `Selesai`, atau `Ditolak`).
4.  **Unduh PDF:** Jika status surat sudah `Selesai`, tombol "Unduh PDF" akan muncul, dan Anda dapat langsung mengunduh surat resmi Anda.
